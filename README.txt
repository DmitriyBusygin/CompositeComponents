Учебный проект для базового изучения технологий: JSF(Composite Component), Bean.
1. Создана JSF страница при помощи составных компонентов, состоящая из index.xhtml и address.xhtml с полями для ввода данных.
2. После нажатия кнопки "Отправить" данные отправляют в управляемый бин AddressBean.java и выводятся на странице подтверждения из управляемого бина.
